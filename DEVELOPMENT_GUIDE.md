# 라이브커머스 솔루션 개발 가이드

이 문서는 사용자가 제공한 상세한 개발 가이드를 포함합니다. 프로젝트의 전체 구조, 기능, 확장 계획을 설명합니다.

## 📋 프로젝트 개요

### 프로젝트명
**Live Commerce** - 실시간 라이브 스트리밍 플랫폼 (라이브커머스 확장 가능)

### 프로젝트 목적
**자체 라이브커머스 솔루션 구축**을 목표로 하는 Next.js 16 기반의 실시간 라이브 스트리밍 플랫폼입니다.

자세한 내용은 프로젝트 루트의 원본 가이드 문서를 참고하세요.

## 🚀 빠른 시작

1. **의존성 설치**
   ```bash
   npm install
   ```

2. **환경 변수 설정**
   - `.env.local` 파일 생성
   - Supabase 및 YouTube API 키 설정

3. **데이터베이스 설정**
   - Supabase에서 테이블 생성 (README.md 참고)
   - RLS 정책 설정

4. **개발 서버 실행**
   ```bash
   npm run dev
   ```

## 📚 주요 기능

- ✅ 실시간 라이브 방송 통합
- ✅ 크리에이터 랭킹 시스템
- ✅ 통합 플레이어
- ✅ 관리자 백오피스
- ✅ 광고 관리 시스템
